03 · Funzionamento

Flusso operativo dell'agente

Dalla richiesta dell'utente al report finale, oppure alla Pull Request quando viene rilevato un bug.

Punti chiave del flusso

  • Esito positivo — se tutti i test passano, l'agente chiude il run con un report finale.
  • Esito con bug — se un test fallisce, l'agente investiga la causa, propone la correzione e apre una Pull Request.
  • Tracciabilità end-to-end — ogni passo è loggato in tempo reale e ogni fix viene validato con dati reali prima di essere proposto.
