ETL Agent Tester

Un agente autonomo che testa le pipeline ETL su BigQuery, individua i bug nel codice SQL e propone le correzioni come Pull Request — end to end.

Cosa fa

  • Pianifica una batteria di test di qualità sull'ETL.
  • Esegue ogni test su BigQuery e ne interpreta gli esiti.
  • Individua la causa nel codice SQL quando un test fallisce.
  • Propone la correzione aprendo una Pull Request su GitHub.

Caratteristiche distintive

  • Autonomia operativa — un solo input dell'utente, l'agente porta a termine il flusso.
  • Tracciabilità completa — ogni passo è loggato e visibile in tempo reale.
  • Integrazione cloud-native — Vertex AI, BigQuery, Cloud Run, GitHub.
  • Human-in-the-loop — la fix è proposta, mai applicata senza review.
