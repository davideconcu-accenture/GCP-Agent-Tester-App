VEA

Verifica ETL Agentica

Un'app agentica per il testing autonomo di pipeline ETL su BigQuery: pianifica i test, li esegue, individua i bug nel codice SQL e propone le correzioni come Pull Request.

01

Use case

Costruire un agente AI capace di creare casi di test per le pipeline ETL su BigQuery. Una fase oggi prevalentemente manuale, ripetitiva e ad alto rischio di errore.

02

Idea

Massima autonomia all'agente: pianificazione dei test, esecuzione su BigQuery, individuazione dei bug e proposta della fix come Pull Request. Un solo input dell'utente, workflow end-to-end.

03

Implementazione

In due parole: app agentica. LLM Gemini su Vertex AI, framework Google Agent Development Kit, tool Python per agire su BigQuery e GitHub.

04

Limitazione dei dati

Niente clienti reali a disposizione: ambiente mock con dataset BigQuery fittizi, ETL di esempio e repository dedicato. Perimetro di demo — l'architettura è pronta per casi reali.
