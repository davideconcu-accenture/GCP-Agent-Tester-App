02 · Infrastruttura

Architettura del sistema

Tre componenti distinti: i dati su BigQuery, l'app agentica su Cloud Run, il codice SQL versionato su GitHub.

Dati

BigQuery

  • Tabelle sorgente, staging e finali nel data warehouse.
  • Accesso in sola lettura da parte dell'agente.
  • Autenticazione via service account con permessi minimi.
Applicazione

Cloud Run

  • Container unico con frontend e backend integrati.
  • Scaling automatico, HTTPS gestito dalla piattaforma.
  • Integrazione con Vertex AI tramite service account.
Codice

GitHub

  • Repository SQL versionato, una cartella per ETL.
  • Token gestito tramite Secret Manager.
  • Pull Request automatica, merge sempre umano.
